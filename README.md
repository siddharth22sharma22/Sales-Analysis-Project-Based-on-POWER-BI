# End-to-End Sales Performance Analytics Pipeline

## 🚀 Project Overview
An end-to-end data analytics project that processes raw sales data, performs feature engineering, and delivers an interactive Power BI dashboard for business intelligence.

## 🛠️ Tech Stack & Workflow
1. **Data Modeling:** Powe Bi Transform Features used to handle missing values, anomalies, and structural formatting.
2. **Feature Engineering:** Extracted and aggregated core metrics to calculate top/bottom performing product performance.
3. **Data Modeling & Visualization:** Power BI used to design an executive dashboard tracking Sales, Quantity, and Profit metrics using a high-contrast visual matrix.

## 📊 Key Insights Captured
* Identified top 5 revenue-generating products (Led by Apple iPhone 14).
* Isolated bottom 5 underperforming products by profit margins to flag operational bottlenecks.
* Analyzed volume vs. profitability discrepancies across product lines.
* Sales Over Period of time
* Comparision of Sales/Profit/Quantity Between TWO DATES
* Map Structural Representation by Sales per City
